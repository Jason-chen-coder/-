# MicroWebApp-vue

本项目结合了qiankun框架和NPM形式进行微前端的初步实践;

项目启动方法:

1.启动子应用:qiankun-app

2.启动父应用:parent-app

说明:npmapp和npm-page-app是用于开发依赖包的项目;参考项目:feui

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
